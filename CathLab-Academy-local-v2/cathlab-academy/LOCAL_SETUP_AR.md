# تشغيل Cath Lab Academy محليًا

هذه النسخة تعمل كواجهة تعليمية محلية مبنية بـ React وVite. لا تحتاج قاعدة بيانات لتجربة المسارات، قوائم الكفاءات، المحاكاة، أو معاينة الشهادة.

## المتطلبات

- Node.js `20.19` أو أحدث، أو Node.js `22.12` أو أحدث. يوصى باستخدام Node.js 22 LTS.
- اتصال بالإنترنت في أول تشغيل فقط لتنزيل الحزم.
- متصفح حديث مثل Chrome أو Edge.

تحقق من الإصدار في PowerShell:

```powershell
node --version
npm --version
```

## التشغيل على Windows

افتح PowerShell داخل المجلد الذي تريد حفظ المشروع فيه، ثم نفّذ:

```powershell
git clone https://github.com/malsoufyani95/Ms-cathlab-academy.git
cd Ms-cathlab-academy
npm ci
npm run dev
```

افتح الرابط الذي يظهر في النافذة، وغالبًا سيكون:

```text
http://localhost:5173
```

صفحة معلومات برنامج الدبلوم:

```text
http://localhost:5173/program.html
```

لا تفتح `index.html` بالنقر المزدوج؛ يجب تشغيله من خلال الأمر `npm run dev`.

## فتح المنصة من جوال على الشبكة نفسها

الأمر الحالي يتيح الوصول من الشبكة المحلية. بعد تشغيل `npm run dev`، استخدم عنوان `Network` الذي يظهر في PowerShell، مثل:

```text
http://192.168.1.20:5173
```

يجب أن يكون الكمبيوتر والجوال على شبكة Wi-Fi نفسها. إذا لم يفتح الرابط، اسمح لـ Node.js بالاتصال عبر Windows Firewall للشبكات الخاصة فقط.

## اختبار نسخة الإنتاج

```powershell
npm run check
npm run preview
```

يُنشئ البناء مجلد `dist`. افتح رابط المعاينة الذي يظهر في PowerShell.

## حلول سريعة للمشكلات

إذا كان المنفذ `5173` مستخدمًا:

```powershell
npm run dev -- --port 5174
```

إذا ظهرت مشكلة في ذاكرة تنزيل npm:

```powershell
npm ci --cache .npm-cache --prefer-online
```

إذا كان أمر `node` غير معروف، ثبّت Node.js 22 LTS ثم أغلق PowerShell وافتحه من جديد.

## حدود النسخة المحلية

- بيانات التقدم محفوظة في المتصفح على الجهاز نفسه فقط.
- تسجيل الدخول وقاعدة البيانات ورفع الملفات غير متصلة بعد.
- الشهادة معاينة تعليمية، ولا تصبح اعتمادًا رسميًا إلا بعد ربط اعتماد المدرب وحوكمة القسم.
